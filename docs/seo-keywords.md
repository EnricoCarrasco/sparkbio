## Keywords to AVOID

| Keyword | Why |
|---------|-----|
| free link in bio | No free tier — only 7-day trial |
| cheapest link in bio | Damages premium positioning |
| link tree clone | Pejorative, signals copycat |
| social media dashboard | Confuses with Buffer/Later |
| free linktree alternative | Contradicts paid model |

## Blog Content Strategy

**English focus:** Design, features, professional branding, analytics
**Portuguese focus:** Monetization, selling, affiliate marketing, growth

Blog posts should NOT be 1:1 translations. Write original content per market.

**Keyword density targets:**
- Primary keyword: 0.8-1.2%
- Secondary keywords: 0.3-0.6% each
- LSI keywords: natural distribution
